# 📚 Student Record System in C++

## 📌 Project Description
This is a simple Student Record System developed in C++ using a structure.  
It allows the user to input student details such as ID, name, course code, course name, and marks, and then displays the stored information.

---

## 🚀 Features
- Take student details as input
- Store data using `struct`
- Display complete student record
- Simple console-based interface

---

## 🧑‍💻 Technologies Used
- C++
- iostream library
- Basic input/output handling

---

## 📂 Program Structure
The program uses a `Student` structure with the following fields:
- Student ID
- Student Name
- Course Code
- Course Name
- Course Marks

---

## ▶️ How to Run
1. Open the project in any C++ IDE (Eclipse / Code::Blocks / Visual Studio)
2. Compile the program
3. Run the executable file
4. Enter required student details in console
